# 太虚归一

## 故事背景

六十年后，大宋武林风云变幻。昔日天龙三杰（段誉、虚竹、萧峰）的传说已成为江湖人口中的旧事，而他们留下的绝学与恩怨，正悄然催生新的劫数。

大理段氏凭借六脉神剑与一阳指，成为武林正道之首；少林派稳居武林正宗；逍遥派早已隐匿江湖，但其邪功遗毒未消。

而真正的风暴，来自一个神秘少年——**段轩**（段誉的曾孙），他体内同时流淌着大理皇室、北冥神功与萧峰临终托付的武功血脉，却对自己的身世一无所知。

## 大纲

### 第一卷：太虚之始（共30章）

**主线**：段轩下山寻根，卷入大理皇室与少林寺的三十年旧怨

- 第1-3章：段轩在无量山长大，拜师逍遥派旁支，意外获得北冥神功残卷
- 第4-10章：下山历练，结识少女**阿蘅**（虚竹后人，灵鹫宫新一代圣女），发现她身负逍遥派掌门信物
- 第11-20章：大理皇室血案，段轩被指为凶手，逃亡途中发现自己的皇室血统；少林寺罗汉阵现身，暗示三十年前的隐秘
- 第21-30章：真相揭露——段誉晚年将萧峰毕生武功封存于无量山玉壁，等待有缘人；段轩被选为"太虚归一"的承载者，必须在一年内集齐三脉才能化解即将到来的"武林浩劫"

**核心主题**：血脉与命运、传承与放下

### 第二卷：三脉合一（共30章）

**主线**：段轩与阿蘅携手，寻找萧峰、段誉、虚竹三位前辈遗留的三件信物，化解浩劫

- 第31-50章：萧峰遗脉在雁门关外，段轩北上遇到辽国残部；阿蘅发现灵鹫宫旧址石壁上的秘密
- 第51-60章：大理皇宫变故，段正明晚年将六脉神剑图谱托付；虚竹遗物现身少林藏经阁

**核心主题**：三脉合一，天下太平

### 第三卷：归一之战（共30章）

**主线**："武林浩劫"真相——逍遥派掌门无崖子假死复生，意图以"太虚之力"一统江湖

- 第61-75章：无崖子现身，揭露他才是真正的"太虚归一"计划的主谋，六十年前假死逃避
- 第76-85章：段轩集三脉之力，却发现自己体内的"杂血"是双刃剑——既能救世，也能灭世
- 第86-90章：最终决战——不是武力的较量，而是对"我是谁"的终极追问

**核心主题**：无我，方能归一

## 武功体系

- **北冥神功**：逍遥派至高内功，段轩主要武功
- **六脉神剑**：大理皇室绝学，远程剑气攻击
- **降龙十八掌**：萧峰绝学，刚猛无比
- **天山折梅手**：灵鹫宫绝学，化解一切招式
- **太虚归一**：终极武学，融汇三脉而成
